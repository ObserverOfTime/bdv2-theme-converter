bdv2-theme-converter

Converts BDv2 themes to legacy BD themes and Stylus UserStyles.

Installation

npm

$ npm install --save ObserverOfTime/bdv2-theme-converter

yarn

$ yarn add ObserverOfTime/bdv2-theme-converter
